# Requirement: Project Invite Permissions

Admins need to invite members to a specific project and choose a project role: viewer, editor, or manager.

Acceptance:

- Managers can invite users to their project.
- Editors and viewers cannot invite users.
- Invited users receive a pending invitation.
- Accepting an invitation adds membership with the selected role.
- The project member list updates after acceptance.

Out of scope:

- Billing seat limits.
- Real email delivery.
